# Customer Segmentation using Data Mining

## Project Overview
This project performs customer segmentation using K-Means clustering.

## Technologies Used
Python
Pandas
Scikit-learn
Matplotlib
Google Colab

## Objective
To group customers based on their spending behavior and income.

## Steps
1. Data preprocessing
2. Elbow method to find optimal clusters
3. K-Means clustering
4. Visualization of customer groups

## Output
Customers are segmented into different groups such as high-value and low-value customers.
